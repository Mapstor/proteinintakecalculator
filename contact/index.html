Troubleshooting Common Issues

Try these fixes before contacting us:

Calculator shows "NaN" or no result

Make sure you've entered a valid number in the weight field. Remove any non-numeric characters (letters, symbols). Try refreshing the page and re-entering your values.

Results seem too high or too low

Double-check your weight units (lbs vs kg). Verify you've selected the correct activity level -- most people overestimate theirs. Make sure you're using the right calculator tab for your goal.

Page not loading or layout broken

Try a hard refresh (Ctrl+Shift+R or Cmd+Shift+R). Clear your browser cache. The calculator requires JavaScript -- make sure it's enabled in your browser settings.

Can't switch calculator tabs

This is usually a JavaScript issue. Try disabling browser extensions that might block scripts (ad blockers sometimes interfere). Try a different browser.

Print not working properly

First calculate your results, then use the print button on the results card. If that doesn't work, use your browser's built-in print function (Ctrl+P or Cmd+P).

Mobile display issues

Try rotating your device to landscape mode for tables. If the calculator is cut off, try zooming out. Clear your browser cache if styles look broken.

The calculator is designed to give you a solid starting point that works for most people. For personalized recommendations, consult a registered dietitian.
